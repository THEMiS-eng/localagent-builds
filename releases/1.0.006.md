# Build 1.0.006

**Released:** 2026-02-01

## What's New

### Modular Build System
LocalAgent now uses a modular architecture where each component has its own version. Updates can be tracked per-module.

### Build Check API
- `GET /api/build/check` - Compare local vs GitHub versions
- `GET /api/build/modules` - List installed modules

### UI Improvements
- Landing page displays BUILD version (e.g., v1.0.006)
- Banner shows "X module update(s) available"
- Release notes modal shows per-module changes

## Updated Modules

| Module | Version | Changes |
|--------|---------|---------|
| localagent-service | 3.3.032 | Build check endpoints |
| themis-ui | 0.1.7 | Build version display |

## Installation

```bash
cd ~/Downloads
unzip -o localagent-build-1.0.006.zip -d ~/localagent-modular
cd ~/localagent-modular && ./INSTALL.sh
./START.sh
```
